# lifx-ctrl

Command line tool for dealing with LIFX light bulbs.

```
Usage: ./lifx-ctrl.rb options
    -o, --off                        switch light off
    -i, --info                       get info
    -c, --color color                set color (red|green|blue|white|orange|yellow|cyan|purple|pink)
    -h, --hue hue                    set hue (0..360)
    -l, --label label                set label of device to be controlled
    -s, --saturation saturation      set saturation (0..1)
    -b, --brightness brightness      set brightness (0..1)
    -k, --kelvin kelvin              set kelvin (2500..9000)
        --help                       show this message
```
